Task 5 – Data Analysis on CSV Files

Objective

Analyze a CSV dataset using Python and Pandas to extract useful insights and visualize the data.

Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook / Google Colab

Project Files

- "Task5_Data_Analysis.ipynb" – Jupyter Notebook containing the analysis
- "sales_data.csv" – Dataset used for analysis
- "charts/" – Generated charts and graphs
- "README.md" – Project documentation

Tasks Performed

- Imported the CSV file using Pandas.
- Loaded the data into a DataFrame.
- Checked the dataset using "head()", "shape", and "info()".
- Cleaned missing values (if any).
- Grouped data using "groupby()".
- Calculated summary statistics.
- Created bar charts to visualize the results.
- Drew conclusions from the analysis.

Key Concepts

- Pandas
- DataFrame
- CSV File Handling
- Data Cleaning
- GroupBy
- Data Visualization

Outcome

Successfully analyzed the CSV dataset and created charts to understand the data better using Python and Pandas.
